For production use, replace this with your actual app icon:
- Create a 512x512 PNG image
- Name it "icon.png"
- Place it in this assets folder

The icon will be used for:
- Windows .exe installer
- Mac .dmg installer
- Linux .AppImage
- App window icon
- Taskbar/dock icon

Recommended tools for creating icons:
- Figma (free)
- Canva (free)
- Adobe Illustrator
- Inkscape (free)
